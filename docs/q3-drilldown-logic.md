# Q3 下钻逻辑梳理

> 归属文档：洋葱学园 · 归因分析 SOP v2.1  
> 适用步骤：Q3 定位下钻

---

## 核心思路

**先收窄问题范围，再决定往哪钻**，不是上来就无脑拆维度。

---

## 第一步：判断问题是全量还是局部

这一步决定后面所有方向，跳过它就等于没做 Q3。

快速跑一张各维度变化率表（学段、渠道各跑一遍），看有没有**逆势维度**：

```
所有维度同向跌，差距 < 5%  →  全量问题
有维度逆势涨，差距 > 10%   →  局部问题
```

**全量问题**：不要急着下钻。大概率是季节性退潮、大促后回落、外部环境，重点看时段分布和同比，找到"退潮节点"就够了。

**局部问题**：进入第二步，用对比定位哪里有个性问题。

---

## 第二步：多维对比锁定跌点（局部问题专用）

**核心思路是"对比定性"，不是"贡献度排名"。**

找的不是"谁贡献最大"，而是"哪里和其他维度明显不一样"——某维度跌但其他正常，就是那里有个性问题。

四个对比维度，按顺序优先级依次看：

### A. 年级 × 渠道（最优先）

最常用的交叉，一张表同时看学段和渠道的分化：

| 发现 | 定性 |
|---|---|
| 初中所有渠道都跌，小学/高中正常 | 初中学段有个性问题，与渠道无关 |
| 所有学段商业化都跌，电销正常 | 商业化投放问题，与学段无关 |
| 只有初中×商业化跌 | 两个维度交叉的特有问题 |
| 所有学段所有渠道都跌 | 重新考虑是否全量问题，回第一步 |

### B. 用户分层（定位流失是哪类用户）

| 用户层 | 跌了说明 |
|---|---|
| 新用户首购跌 | 获客/落地页/转化链路问题 |
| 老用户续费跌 | 产品/服务质量问题 |
| 高净值用户跌 | 高价课或套餐策略问题 |
| 流失召回跌 | 私域运营/促销策略问题 |

### C. 公私域（流量来源结构有没有变）

| 跌的来源 | 说明 |
|---|---|
| 公域自然流量跌 | 平台算法/搜索排名问题 |
| 私域推送跌 | 运营/触达率问题 |
| 付费投放跌 | ROI/创意/落地页问题 |

### D. 渠道横向对比（判断是共性还是个性）

- 电销涨 + APP 跌 → APP 端有独立问题
- 所有渠道都跌 → 产品或外部原因，不是渠道问题
- 只有商业化电商跌 → 电商平台特有问题

---

## 第三步：专项深挖（锁定跌点后才进）

多维对比做完，跌点已经明确，这时候才进专项。**没有锁定跌点就进专项，是最容易浪费时间的错误。**

### 专项 A：学段有问题 → 下钻到 SKU

按 `good_name × mid_grade` 拉明细，找 TOP 下跌 SKU，然后判断原因类型：

| 原因 | 验证方法 |
|---|---|
| 季节性退潮 | 对比去年同期同 SKU，若同样跌则是自然周期 |
| 曝光骤降 | 查 show UV 有没有骤降（下架/减曝光） |
| 价格调整 | 对比两周 sub_amount 均值 |
| 内容质量 | 查完课率/差评率 |

### 专项 B-1：APP/商业化有问题 → 四层漏斗

```
曝光 UV → 点击 UV → 试听 UV → 支付 UV
```

逐层算转化率，找断点在哪一层。同时拆 iOS vs Android，判断是否单平台问题。

### 专项 B-2：电销有问题 → 团队三步归因

1. 按团队 × 学段拆 GMV，找跌的团队
2. 拆线索量 vs 转化率：线索少是获客问题，线索多转化低是销售/商品问题
3. 对比团队人效，排查话术/激励/培训

---

## 整体决策树

```
Q3 入口
    ↓
全量 vs 局部判断
    ├── 全量问题 → 看时段分布和同比 → 找退潮节点 → 进 Q4
    │
    └── 局部问题
            ↓
        年级 × 渠道对比
            ↓
        用户分层对比（需要时）
            ↓
        公私域 / 渠道横向对比（需要时）
            ↓
        锁定跌点
            ↓
        ┌── 学段有问题 → 专项A：SKU 下钻
        ├── APP 渠道有问题 → 专项B-1：漏斗
        └── 电销有问题 → 专项B-2：团队归因
            ↓
           进 Q4
```

---

## 最容易犯的两个错误

1. **跳过全量/局部判断直接下钻**：把季节性退潮归因成渠道问题，分析半天结论是错的
2. **多维对比没做完就进专项**：比如只看了学段，没发现电销逆势上涨，误以为是学段个性问题，实际是商业化问题
